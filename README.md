# Three_In_A_Row
## Visuals/Audio
* **Audio System:** Background Musi через кастомний `MusicManager`. Додано можливість вмикати/вимикати музику через UI-чекбокс (Toggle) у головному меню зі збереженням налаштувань гравця між сесіями гри.
* **VFX (Ефекти):** Використання `Particle System` для візуалізації знищення (матчу) елементів на ігровому полі. Кастомні спрайти для візуального виділення бонусних елементів (стрілки, бомби).

## UI
* **UI:** Повністю побудована логіка переходів через центральний `GameManager` (Головне меню, Вибір режиму, Вибір рівня).
* **Pause UI:** Реалізовано екран паузи, екран перемоги з відображенням рахунку, екран поразки та панель фінальних результатів для аркади.
* **Leaderboard UI:** Додано поле вводу (`Input Field`) для реєстрації імені гравця перед стартом та панель виводу рейтингу (Топ-5). Без вводу імені гру неможливо почати

## Game Modes & Levels
* **Arcade:** Гра на виживання з таймером. Мета гравця — набрати максимальну кількість балів доки не сплине час (наприклад, 2 хвилини), щоб потрапити в таблицю рекордів.
* **Levels:** Проходження рівнів на кількість ходів. Кожен рівень має свою унікальну форму ігрового поля (налаштовується через `ArrayLayout`), ліміт ходів (`Moves`) та цільовий рахунок (`Target Score`). Прогрес розблокування рівнів зберігається локально.

## Custom Mechanics
* **Бонуси (Power-ups):** Механіки винагород за з'єднання 4-х, L-подібних та 5-x елементів:
  * *Arrow (Лінійна стрілка):* Знищує всі елементи у відповідній горизонтальній або вертикальній лінії (їх два види).
  * *Bomb (Бомба):* Знищує сусідні клітинки навколо себе у заданому радіусі.
  * *Color Bomb (Кольорова бомба):* Повністю знищує всі елементи обраного типу на всьому ігровому полі.
* **Leaderboard System:** Система відстеження рекордів в аркадному режимі. Локальне збереження ТОП-5 гравців (через `PlayerPrefs` та серіалізацію в `JSON`). Система автоматично перевіряє, чи покращив гравець свій попередній результат, та сортує список від найбільшого до найменшого.
